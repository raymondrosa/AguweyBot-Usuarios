# 🤖 AguweyBot - Asistente Multi-Modelo IA

Asistente inteligente con soporte para Mistral, Qwen 2.5 72B y Gemma 4 31B.

## ✨ Características
- 📄 Análisis de documentos (PDF, Excel, CSV, TXT, DOCX)
- 🎤 Texto a voz en español
- 💾 Guardado y carga de conversaciones
- 🧠 3 modelos IA disponibles
- 🌐 Desplegado en Streamlit Cloud

## 🔧 Configuración Local

```bash
pip install -r requirements.txt
streamlit run app.py
